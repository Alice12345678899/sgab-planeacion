## Integrantes

## Descripción del proyecto
Este repositorio contiene la planeación inicial del proyecto *Sistema de Gestión Académica Básico (SGAB)*.  
El sistema está orientado a la administración de alumnos, materias, calificaciones y usuarios dentro de un entorno académico.

## Objetivo del repositorio
El objetivo de este repositorio es documentar el análisis previo al desarrollo del sistema, aplicando los fundamentos de la Ingeniería de Software: calidad, estimación, riesgos y alcance, sin incluir implementación de código.
